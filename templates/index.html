Project Information

This is a simple To-Do app built using Flask, a Python web framework. It allows you to add, delete, and update tasks in a to-do list.

Steps to Create and Deploy

  1. Clone this repository to your local machine.
  2. Install Flask and any required dependencies by running pip install -r requirements.txt.
  3. Create a virtual environment for your project.
  4. Run the Flask app locally using flask run.
  5. Sign up for a Vercel account at [vercel.com](https://vercel.com/).
  6. Deploy your Flask app on Vercel by following the deployment instructions. Deploy it from your Main branch
